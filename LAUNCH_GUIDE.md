## Quick Start: 3 Easy Steps

### Step 1: Deploy to Vercel (Recommended)
```bash
# 1. Connect your GitHub account to Vercel
# 2. Import this repository
# 3. Click "Deploy"
# That's it! Your site is live in seconds.
```

**Benefits:**
- Free SSL/HTTPS
- Custom domain
- Automatic backups
- CDN distribution
- Environment variables
- Analytics included

### Step 2: Configure Custom Domain
1. Register domain (anilkataria.com)
2. In Vercel project settings → Domains
3. Add your custom domain
4. Update DNS records (Vercel provides instructions)
5. Done! Site loads on your domain

### Step 3: Add Contact Email Forwarding
Set up email service to handle contact form submissions. Options:
- Sendgrid (free tier available)
- AWS SES
- Mailgun
- Or any email service provider

- Update: Phone, email, WhatsApp

**Credentials & Certificates**:
- File: `data/certificates.ts`
- Update: Add/modify certificates as needed

**Student Testimonials**:
- File: `data/testimonials.ts`
- Update: Add new student feedback

**Courses**:
- File: `data/courses.ts`
- Update: Course details and descriptions

**FAQ**:
- File: `data/faq.js`
- Update: Common questions and answers

**Branding Colors**:
- File: `app/globals.css`
- Update: Color variables (primary, secondary, accent)

### Add New Pages
1. Create folder: `/app/new-page`
2. Create file: `/app/new-page/page.tsx`
3. Add navigation link in `components/common/header.tsx`
4. Add footer link in `components/common/footer.tsx`

## Troubleshooting

### Common Issues

**Issue**: Website is slow
- **Solution**: Check Vercel analytics, enable caching, optimize images

**Issue**: Form not sending emails
- **Solution**: Set up email service (Sendgrid, AWS SES, etc.)

**Issue**: Images not loading
- **Solution**: Check image paths in `/public/images/branding`

**Issue**: Mobile menu not working
- **Solution**: Clear browser cache, test in incognito mode

**Issue**: Domain not resolving
- **Solution**: Check DNS records in Vercel settings, wait 24 hours for propagation
